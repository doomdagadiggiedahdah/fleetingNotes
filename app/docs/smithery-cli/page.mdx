This guide will walk you through installing the Smithery CLI on Windows using Scoop.
## Installation Steps
### <span className="text-foreground">Step 1:</span> Install Scoop
For more information about Scoop, visit [the official Scoop GitHub repository](https://github.com/ScoopInstaller/Scoop).<br />
In Windows PowerShell:

<CodeBlock 
  language="powershell" 
  code="Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser"
  showHeader={true}
  className="mb-4"
/>

<CodeBlock 
  language="powershell" 
  code="Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression"
  showHeader={true}
  className="mb-6"
/>

### Step 2: Add Scoop to your PATH

<CodeBlock 
  language="powershell" 
  code="$oldPath = [Environment]::GetEnvironmentVariable('Path', 'User')"
  showHeader={true}
  className="mb-4"
/>

<CodeBlock 
  language="powershell" 
  code='$newPath = "$oldPath;$env:USERPROFILE\scoop\shims"'
  showHeader={true}
  className="mb-4"
/>

<CodeBlock 
  language="powershell" 
  code="[Environment]::SetEnvironmentVariable('Path', $newPath, 'User')"
  showHeader={true}
  className="mb-6"
/>

### Step 3: Restart Powershell

### Step 4: Check Scoop installation

<CodeBlock 
  language="powershell" 
  code="scoop --version"
  showHeader={true}
/>

### Step 5: Install Smithery

<CodeBlock 
  language="powershell" 
  code="scoop bucket add smithery https://github.com/smithery-ai/scoop-smithery"
  showHeader={true}
/>

<CodeBlock 
  language="powershell" 
  code="scoop install smithery"
  showHeader={true}
/>

### Step 6: Check smithery installation

<CodeBlock 
  language="powershell" 
  code="smithery --help"
  showHeader={true}
/>

### Step 7: Install and Run Server

Install the server:
<CodeBlock 
  language="powershell" 
  code="smithery install @smithery-ai/server-sequential-thinking --client claude"
  showHeader={true}
/>

Run the server in cursor MCP settings:
<CodeBlock 
  language="powershell" 
  code='smithery run @smithery-ai/server-sequential-thinking --config "{}"'
  showHeader={true}
/>

If the above command doesn't work, try using the full Scoop shim path:
<CodeBlock 
  language="powershell" 
  code='C:\Users\YourUsername\scoop\shims\smithery.exe run @smithery-ai/server-sequential-thinking --config "{}"'
  showHeader={true}
/>
Note: Replace `YourUsername` with your actual Windows username.

Enjoy!
